# ESPHome Ducky

Control a virtual keyboard from HomeAssistant using an ESP32 connected to an [Adafruit KB2040](https://www.adafruit.com/product/5302).
This project is conceptually based on, and named after, [esphome_ducky](https://github.com/jensweimann/esphome_ducky) but does not (yet) contain any copied code.
This project is also very much a work in progress, and the current contents of the repository are little more than a technical demo.
